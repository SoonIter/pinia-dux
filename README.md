# Pinia-dux

[中文](./README.md) | [English](./README.en-US.md)

一个 piniaAPI 风格的 react 状态管理库。

用 Vue pinia 的 api 写 React

目前支持持久化，和多页面状态同步。

```shell
npm i piniadux@latest
```

```shell
yarn add piniadux@latest
```

```shell
pnpm add piniadux@latest
```
